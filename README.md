# 🐍 MEDUSA - Plataforma de Orientación y Denuncia

**Estado del Proyecto:** En desarrollo activo (Fase MVP) 🚀
**Proyecto Final:** "Ciudadanía en Acción" - YOUTH LAB CREO

## 📖 Sobre el Proyecto
MEDUSA es el Eje 1 de una estrategia nacional para reducir la incidencia y el subregistro de la violencia de género en El Salvador. Esta Progressive Web App (PWA) está diseñada para ofrecer:
- Chatbot inteligente para orientación anónima.
- Rutas personalizadas de atención y denuncia.
- Herramientas de preservación de evidencia digital.

## 🛠️ Stack Tecnológico (Proyectado)
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Backend/Framework:** Laravel / PHP (Fase 2)
- **Base de Datos:** MySQL
- **Integraciones:** API de Geolocalización, IA Chatbot
